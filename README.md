This repository is used as a portfolio to demonstrate skills in working with mobile app development tools for data collection, such as KoboToolBox/ODK, specifically XLSForm for creating surveys. This repository contains XLS forms and external files.
This portfolio is designed to demonstrate skills in working with XLS forms, using ability to set up automatic collect of advanced metadata, and the ability to create different types of questions using skip logic, constraints, groups, repeats, appearance, collecting GPS data, cascade select, calculations, using external files for choices etc.
